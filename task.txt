Missão: Faça uma aplicação para controle de tarefas e envie os links para os repositórios
necessários, os quais devem conter o readme para levantar a(s) aplicação(ões).
A aplicação deve listar as tarefas por ordem de prioridade e estado, além de permitir a inclusão
de uma nova tarefa.
Regras:
* Uma tarefa pode ter diferentes níveis de prioridade (baixa, normal e alta).
* Deve ser possível alterar o estado de uma tarefa (Pendente, Em andamento ou Concluída).
